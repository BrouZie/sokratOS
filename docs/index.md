Welcome to the sokratOS documentation! This comprehensive guide will help you get started with, learn, and master your sokratOS setup.

## Target Audience

This documentation is designed for:

- Students and curious users wanting to learn Linux and tiling window managers
- Developers looking for an efficient, keyboard-driven setup
- Users new to terminal-based workflows
- Anyone interested in customizing their desktop environment

You don't need to be an expert! This documentation assumes you're new to tiling WMs and will guide you through everything step by step.

## Philosophy

sokratOS is designed around these principles:

1. **Keyboard-first**: Everything should be accessible via keyboard shortcuts
2. **Visual clarity**: Beautiful but functional aesthetics
3. **Customizable**: Easy to tweak and make your own
4. **Terminal-centric**: Power through the command line
5. **Learning-friendly**: Documentation and help are always accessible

# Environment Variables Reference

Environment variables used and set by sokratOS.

**Main configuration**: `~/.config/hypr/configs/envs.conf`

## Hyprland Environment Variables

Set in `~/.config/hypr/configs/envs.conf`:

### Display and Wayland

```conf
env = XCURSOR_SIZE,24
env = HYPRCURSOR_SIZE,24
```

**Purpose**: Cursor size for X11 and Hyprland

### XDG Specifications

```conf
env = XDG_CURRENT_DESKTOP,Hyprland
env = XDG_SESSION_TYPE,wayland
env = XDG_SESSION_DESKTOP,Hyprland
```

**Purpose**: Identify desktop environment to applications

### Graphics Backend

```conf
env = GDK_BACKEND,wayland,x11,*
env = QT_QPA_PLATFORM,wayland;xcb
env = SDL_VIDEODRIVER,wayland
env = CLUTTER_BACKEND,wayland
```

**Purpose**: Tell applications to use Wayland when possible

### NVIDIA-specific (if applicable)

```conf
env = LIBVA_DRIVER_NAME,nvidia
env = GBM_BACKEND,nvidia-drm
env = __GLX_VENDOR_LIBRARY_NAME,nvidia
```

**Purpose**: NVIDIA GPU acceleration support

## Shell Environment Variables

Set in `~/.bashrc`:

### PATH Additions

```bash
export PATH="$HOME/.local/share/sokratOS/bin:$PATH"
export PATH="$HOME/.local/bin:$PATH"
```

**Purpose**: Make custom scripts accessible

### Python Environment

```bash
export PYTHONPATH="$HOME/.local/lib/python3.11/site-packages:$PYTHONPATH"
```

**Purpose**: Python module search path

### Neovim Python

```bash
# Set in Neovim config
vim.g.python3_host_prog = "~/.venvs/nvim/bin/python"
```

**Purpose**: Neovim Python provider location

## Installation Variables

Used during installation (in `install.sh`):

```bash
export REPO_PATH="$HOME/.local/share/sokratOS"
export REPO_INSTALL="$REPO_PATH/install"
```

**Purpose**: Reference paths during setup

## Theme-Related Variables

### Generated by Scripts

```bash
# In sokratos-apply-theme
PRIMARY_HEX=$(<~/.config/sokratOS/matugen/.primary-hex.txt)
COLOR1=$(pastel format name $PRIMARY_HEX)
SHADE=$(...)  # dark or light
```

**Purpose**: Theme color analysis and matching

## Application-Specific

### Tmux

```bash
export TERM=tmux-256color
```

**Purpose**: Enable 256-color support in tmux

### Editor

```bash
export EDITOR=nvim
export VISUAL=nvim
```

**Purpose**: Default editor for git, etc.

### Browser

```bash
export BROWSER=firefox
```

**Purpose**: Default web browser

## Customizing Environment Variables

### Hyprland Variables

Edit `~/.config/hypr/configs/envs.conf`:

```conf
# Add your custom env vars
env = MY_VARIABLE,value
env = ANOTHER_VAR,/path/to/something
```

Then reload:
```bash
hyprctl reload
```

### Shell Variables

Edit `~/.bashrc`:

```bash
# Custom environment variables
export MY_VAR="value"
export PATH="/my/custom/path:$PATH"
```

Then reload:
```bash
source ~/.bashrc
```

## Common Customizations

### Change Default Editor

```bash
# In ~/.bashrc
export EDITOR=nano
export VISUAL=nano
```

### Change Browser

```bash
# In ~/.bashrc
export BROWSER=chromium
```

### Adjust Cursor Size

```conf
# In ~/.config/hypr/configs/envs.conf
env = XCURSOR_SIZE,32
env = HYPRCURSOR_SIZE,32
```

### High DPI Displays

```conf
# In ~/.config/hypr/configs/envs.conf
env = GDK_SCALE,2
env = GDK_DPI_SCALE,0.5
```

### Qt Theme

```conf
# In ~/.config/hypr/configs/envs.conf
env = QT_QPA_PLATFORMTHEME,qt5ct
```

## Viewing Environment Variables

### All Variables

```bash
# List all
env

# Search specific
env | grep PATH
```

### Specific Variable

```bash
echo $PATH
echo $EDITOR
echo $XDG_CURRENT_DESKTOP
```

### In Hyprland

```bash
# View Hyprland env vars
hyprctl getoption env
```
